# Weekly Report Skill

When generating weekly reports, follow these steps:

1. **Gather task records** - Search for completed, in-progress, and pending tasks from the week
2. **Collect meeting notes** - Find and summarize key decisions from meetings
3. **Identify project milestones** - Note any significant achievements or deliverables
4. **Track blockers** - Document any issues that impeded progress
5. **Plan next steps** - Outline priorities and goals for the upcoming week

## Output Format

Structure the weekly report as:
- **This Week's Highlights**: Top 3-5 accomplishments
- **Task Progress**: Categorized by project with status indicators
- **Key Decisions**: Important decisions made during the week
- **Blockers & Risks**: Issues that need attention
- **Next Week's Plan**: Prioritized goals and action items
