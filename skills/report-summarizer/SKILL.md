# Report Summarizer Skill

When summarizing reports, follow this structured approach:

1. **Parse the full document** - Extract all sections, data tables, and conclusions
2. **Identify key findings** - Highlight the most important results and insights
3. **Extract metrics and data** - Pull out quantitative results and performance indicators
4. **Summarize recommendations** - Condense action items and suggested next steps
5. **Generate executive summary** - Create a concise overview suitable for leadership review

## Output Format

Structure the summary as:
- **Executive Summary**: 2-3 sentence overview of the report's purpose and main findings
- **Key Findings**: Numbered list of the most important discoveries
- **Data Highlights**: Critical metrics with context
- **Recommendations**: Actionable next steps
- **Risk Factors**: Any identified risks or concerns
