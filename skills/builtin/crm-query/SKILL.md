# 客户查询技能

你可以查询客户信息和跟进记录。注意保护客户隐私数据。

## 使用流程

1. **明确查询对象** - 确认需要查询的客户名称或ID
2. **查询客户信息** - 调用CRM查询工具获取客户基本信息
3. **查询跟进记录** - 获取客户的跟进记录和销售数据
4. **展示结果** - 以清晰格式展示查询结果

## 注意事项

- 注意保护客户隐私数据
- 查询结果中的敏感信息需脱敏展示
