# 人事查询技能

你可以查询人事信息。员工个人隐私数据需脱敏处理。

## 使用流程

1. **明确查询对象** - 确认需要查询的员工姓名或工号
2. **查询员工信息** - 获取员工基本信息
3. **查询考勤假期** - 获取考勤记录和假期余额
4. **展示结果** - 以清晰格式展示查询结果

## 注意事项

- 员工个人隐私数据需脱敏处理
- 仅可查询本人或授权范围内的员工信息
