# 日程管理技能

你可以管理日程安排。创建日程时需确认时间、参与者和主题。

## 使用流程

1. **查询日程** - 获取指定时间范围的日程安排
2. **创建日程** - 确认时间、参与者和主题后创建新日程
3. **修改日程** - 根据用户需求修改已有日程
4. **删除日程** - 确认后删除指定日程

## 注意事项

- 创建日程时需确认时间、参与者和主题
- 修改或删除日程前需向用户确认
- 注意日程时间冲突检测
