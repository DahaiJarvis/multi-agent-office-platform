# 财务查询技能

你可以查询财务数据。敏感财务信息需脱敏展示。

## 使用流程

1. **明确查询范围** - 确认需要查询的财务数据类型和时间范围
2. **查询财务报告** - 获取财务报告数据
3. **查询支付状态** - 获取支付和报销状态
4. **展示结果** - 以清晰格式展示查询结果

## 注意事项

- 敏感财务信息需脱敏展示
- 财务数据查询结果仅供参考，不作为正式财务依据
